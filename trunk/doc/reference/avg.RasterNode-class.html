Known Subclasses:
Image, VideoBase, Words
Base class for all nodes that have a direct 2d raster representation.
This includes Image, Word, Camera, and Video nodes. RasterNodes can
be rotated. Warping of RasterNodes is implemented using a grid of
reference points. The position of each of these points can be changed.
Properties:
    angle: The angle that the node is rotated to in radians. 0 is
        unchanged, 3.14 is upside-down.
    pivotx, pivoty: The point that the node is rotated around. Default
        is the center of the node.
    maxtilewidth, maxtileheight: The maximum width and height of the
        tiles used for warping. The effective tile size is also
        dependent on hardware and driver limits. (ro)
    blendmode: The method of compositing the node with the nodes under
        it. Valid values are 'blend', 'add', 'min' and 'max'.
