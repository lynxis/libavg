                PanoImage


A panorama image.
Properties:
    href: The source filename of the image.
    sensorwidth: The width of the sensor used to make the image. This value
                 is used together with sensorheight and focallength to
                 determine the projection to use. (ro)
    sensorheight: The height of the sensor used to make the image. (ro)
    focallength: The focal length of the lens in millimeters. (ro)
    hue: A hue to color the image in. (ro, deprecated)
    saturation: The saturation the image should have. (ro, deprecated)
    rotation: The current angle the viewer is looking at in radians.
    maxrotation: The maximum angle the viewer can look at.
