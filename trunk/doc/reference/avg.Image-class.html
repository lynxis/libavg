                    Image


A static raster image on the screen. Images are loaded from files on
program start. Alpha channels of the image files are used as
transparency information.
Properties:
    href: The source filename of the image.
    hue: A hue to color the image in. (ro, deprecated)
    saturation: The saturation the image should have. (ro, deprecated)
