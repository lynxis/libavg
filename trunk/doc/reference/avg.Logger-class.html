Interface to the logger used by the avg player. Enables the setting
of different logging categories and a destination file. Can be used
to log application-specific messages as well.
Each log entry contains the time the message was written, the category
of the entry and the message itself.

Instance Method Details

setCategories(categories) 

Sets the types of messages that should be logged. Possible 
categories are:
    NONE: No logging except for errors.
    BLTS: Display subsystem logging. Useful for timing/performance
          measurements.
    PROFILE: Outputs performance statistics on player termination.
    PROFILE_LATEFRAMES: Outputs performance statistics whenever a
                        frame is displayed late.
    EVENTS: Outputs basic event data.
    EVENTS2: Outputs all event data available.
    CONFIG: Outputs configuration data.
    WARNING: Outputs warning messages. Default is on.
    ERROR: Outputs error messages. Can't be shut off.
    MEMORY: Currently unused.
    APP: Reserved for application-level messages issued by python
         code.
Categories can be or'ed together.

None
