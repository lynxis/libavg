Hic sunt leones

Be warned that this is stuff under development. avg is not polished enough to be easy to install yet. We've had it running on Debian unstable, Gentoo and SuSE boxes, and it shouldn't be impossible to get running on a sane Linux system, but it isn't packaged well yet, so things will hiccup and stutter and spit out compiler errors and in general behave badly until tamed. Of course, at this stage I'm more than happy to help anyone trying to install the beast: coder at c-base.org.

Prerequisites

avg uses a lot of third-party libraries. What follows is a list of packages you'll need. I am not sure that the package names are exact, but the list should enable you to find what's needed. At the moment, avg doesn't check during configure that these libraries are installed, so you'll probably get compiler errors if they aren't.

  • The image format libraries libjpeg62, libpng, libtiff (all optional but highly recommended).
  • libavcodec 0.4.9pre1. Debian users need to add deb ftp://ftp.nerim.net/debian-marillat/ unstable main to /etc/apt/sources.list to get this package. The gentoo package name is ffmpeg.
  • libsdl1.2 (only needed for non-fullscreen and OpenGL support).
  • OpenGL and/or DirectFB (from www.directfb.org. I've used ver. 0.9.19 and several cvs versions of this library. The debian developer package works as well). configure should detect which of the libraries is installed.
  • libxml2.
  • paintlib (from www.paintlib.de. Download version 2.6.2. Make sure the library finds the image format libraries during configure.)
  • libfreetype and pango.
  • Optional: libdc1394 for Firewire digital camera support.

avg itself

Download the tarball: avg-0.1.0.tar.gz.

Unpack it and run ./configure, make, make install


or

do a cvs checkout: 

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/avg login

When prompted for a password for anonymous, simply press the Enter key.

cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/avg co avg

Then ./bootstrap to generate configure.

Now run ./configure, make, make install as usual.
