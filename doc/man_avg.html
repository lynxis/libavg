avg

SYNOPSIS

       avg scriptfile.js [--display=DFB|OGL] [--bpp=16|24]
       [--fullscreen=true|false] [--debug]

DESCRIPTION

       avg plays back an avg media presentation.  avg
       presentations consist of two parts - a file in avg format
       that describes a static media configuration, and a
       javascript file that contains information about the
       dynamics of the presentation. The javascript file is given
       by the first parameter. This file needs to contain
       commands to load and play an avg file. System
       configuration is determined by first reading the
       configuration files /etc/avgrc and ~/avgrc and then
       applying the command-line options.

EXAMPLES

       To play back the presentation trailer.js using the system
       defaults, use:

       avg trailer.js

       To play back the presentation core_control.js in
       fullscreen, using avgs OpenGL display engine, use:

       avg core_control.js --display=OGL --fullscreen=true

OPTIONS

       --display=DFB|OGL
              Set the display engine to use.  DFB selects the
              DirectFB display engine, and OGL selects the OpenGL
              display engine.

       --bpp=16|24
              Set the bits per pixel for the back- and
              frontbuffers. All internal calculations are done
              with 32 bpp - 24 bpp plus 8 bpp transparency -
              regardless of this setting.

       --fullscreen=true|false
              Determines whether playback should happen in
              fullscreen mode (true) or in a window (false). The
              DirectFB engine uses a slow SDL backend in windowed
              mode - this mode is meant for debugging, not
              performance.

       --debug
              Start gdb with the avg player.
