Method Details

clearDataLines(lines) 

Clears the data lines given as argument. Constants to used for these
lines are PARPORTDATA0-PARPORTDATA7. Several of these constants can
be or'ed together to clear several lines. The lines not mentioned in
the parameter are left unchanged.
Returns:
ok

getStatusLine(line) 

Returns the value of one of the parallel port status lines. Possible
lines are STATUS_ERROR, STATUS_SELECT, STATUS_PAPEROUT, STATUS_ACK
and STATUS_BUSY.
Returns:
value

init(DeviceName) 

Opens the given device as a parallel port. If DeviceName is an empty
string, /dev/parport0 is used as device name.
Returns:
None

isAvailable() 

Returns 1 if the parallel port has been opened successfully, 0
otherwise.
Returns:
ok

setAllDataLines(lines) 

Sets and resets all data lines according to the bits set in the
argument. Constants to used for these
lines are PARPORTDATA0-PARPORTDATA7. Several of these constants can
be or'ed together to set several lines. The lines not mentioned in
the parameter are set to 0. Returns 1 if the lines were set,
0 otherwise.
Returns:
ok

setDataLines(lines) 

Sets the data lines given as argument. Constants to used for these
lines are PARPORTDATA0-PARPORTDATA7. Several of these constants can
be or'ed together to set several lines. The lines not mentioned in
the parameter are left unchanged. Returns 1 if the lines were set,
0 otherwise.
Returns:
ok
