play(framerate) 

Opens a playback window or screen and starts playback. framerate is
the number of frames per second that should be displayed. play returns
when playback has ended.
Returns:

screenshot(filename) 

Saves the contents of the current screen in a png file. Returns
1 on success, 0 if the screen couldn't be saved.
Returns:
ok

setDisplayEngine(engine) 

Determines which display backend to use. Parameter can be either
avg.DFB (for DirectFB rendering) or avg.OGL (for OpenGL rendering).
Must be called before loadFile.
Returns:
None

setInterval(time, pyfunc) 

Sets a python callable object that should be executed every time
milliseconds. setInterval returns an id that can be used to
call clearInterval() to stop the code from being called.

id
